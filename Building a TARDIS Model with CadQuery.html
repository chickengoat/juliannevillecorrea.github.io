Building a TARDIS Model with CadQuery

This tutorial shows how to model a simplified TARDIS from Doctor Who usi= ng the Python library CadQuery. The model consists of the main body, roof, = and light, representing the iconic look of the TARDIS.

1. Main Structure: HTML & Canvas

3. Explanation of CadQuery Code for TARDIS

Below is the Python code for constructing the TARDIS in CadQuery. We bre= ak it down step-by-step:

# Import CadQuery
import cadquery as cq

# Dimensions
width =3D 1.0         # Base width of the TARDIS
depth =3D 1.0         # Base depth of the TARDIS
height =3D 2.2        # Total height of the TARDIS
roof_height =3D 0.2   # Height of the roof section
light_height =3D 0.15 # Height of the light
light_diameter =3D 0.1 # Diameter of the light on top

# Step 1: Main Body
body =3D (
    cq.Workplane("XY")
    .box(width, depth, height - roof_height)
)

# Step 2: Roof Structure
roof =3D (
    cq.Workplane("XY")
    .workplane(offset=3Dheight - roof_height)
    .rect(width * 1.1, depth * 1.1)
    .extrude(roof_height / 4)
    .faces(">Z").workplane()
    .rect(width * 1.05, depth * 1.05)
    .extrude(roof_height / 4)
    .faces(">Z").workplane()
    .rect(width * 0.9, depth * 0.9)
    .extrude(roof_height / 2)
)

# Step 3: Light on Top
light =3D (
    cq.Workplane("XY")
    .workplane(offset=3Dheight)
    .circle(light_diameter / 2)
    .extrude(light_height)
)

# Step 4: Assemble the Model
tardis =3D body.union(roof).union(light)

Explanation of Each Step

  1. Main Body: We create the TARDIS body using .bo= x() to define a rectangular prism.
  2. Roof Structure: The roof is modeled as stacked recta= ngles that decrease in size as they go up. The code uses multiple layers to= mimic the stepped roof.
  3. Light on Top: The light on top is a small cylinder c= reated using .circle() and .extrude().
  4. Assembly: We use union() to join each p= art into a single TARDIS model.
